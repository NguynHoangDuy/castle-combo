# Castle Combo - Complete Cards Database

---

## Card Structure Overview

Every card in Castle Combo has these attributes:

| Attribute | Description |
|-----------|-------------|
| **Name** | Character name |
| **Type** | Castle (silver) or Village (brown) |
| **Cost** | Gold required to buy |
| **Shields** | 1-2 colored shields (6 colors) |
| **Ability** | Immediate effect when placed |
| **Scoring** | End-game points condition |
| **Image** | Character artwork |

## Ability Types

### Immediate Abilities (When Placed):

| Ability | Effect | Example Cards |
|---------|--------|---------------|
| +Gold | Gain gold coins | Woodcutter, Baker, Merchant |
| +Keys | Gain key tokens | Knight, General, Militia |
| Discount | Reduce cost of future cards | Baron, Duke, Nobleman |
| Move Messenger | Switch active row free | Wizard |
| Look at Deck | View upcoming cards | Fortune Teller |

### Scoring Conditions (End Game):

| Condition | Points For | Example Cards |
|-----------|------------|---------------|
| Per Shield Color | Each shield of specific color | King (+3 per Blue) |
| Per Card Type | Castle or Village cards | Steward (+1 per Castle) |
| Per Adjacent | Cards next to this one | Prince (+2 per adjacent Castle) |
| Per Row/Column | Cards in same line | Guard (+1 per column) |
| Per Diagonal | Cards diagonal to this | Archer (+2 per diagonal) |
| Set Collection | Combinations of shields | Queen (+10 per Blue+Green+Orange set) |
| Condition Check | If condition is met | Barbarian (+10 if no Green) |
| Per Position | Specific grid positions | Butler (+2 per corner) |
| Resources | Remaining gold/keys | Treasurer (+1 per 2 Gold) |
